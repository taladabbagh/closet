# Online Closet

## Vision

A premium digital wardrobe that allows users to:

- Store clothing
- Categorize clothing
- Build outfits
- Save outfits
- Plan vacations
- Generate packing lists
- Track wear count
- Track laundry
- Create custom filters
- Eventually include AI outfit recommendations
